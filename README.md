# Terminal de Suporte Proativo

## Descrição
Aplicação em C# (console) que simula um terminal de diagnóstico de rede, com foco em usabilidade e prevenção de erros.

## Funcionalidades
- Pingar um IP
- Formatar unidade com confirmação
- Menu de comandos na tela
- Comando de ajuda (help ou ?)

## Heurísticas aplicadas
- Prevenção de erros: confirmação antes de formatar
- Reconhecimento em vez de recordação: menu visível
- Ajuda e documentação: comando help

## Como executar
dotnet run
